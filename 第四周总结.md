### 点亮LED灯

要把LED的GPIO引脚PB0配置成输出模式，即配置GPIO的端口配置低寄存器CRL

```c
//清空控制PB0的端口位
GPIOB_CRL &=~(0x0f<<(4*0));
//配置PB0为通用推挽输出
GPIOB_CRL |=  (1<<(4*0));

```

**&=~ 将前值的位取零**     **|= 用于配置输出模式 且不影响寄存器的其他位**

然后将PB0位输出设置为低电位

```c
//PB0输出低电平
GPIOB_ODR |= (0<<0);
```

最后一步，打开APB2的外设时钟，使用RCC

```c
//开启GPIOB的端口时钟
RCC_APB2ENR |= (1<<3);
```
